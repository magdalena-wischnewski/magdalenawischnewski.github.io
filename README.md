# Dr. Magdalena Wischnewski — Portfolio Website

Static portfolio website for GitHub Pages.

## Recommended repository name

For the cleanest website URL, create the repository with this exact name:

```text
magdalenawischnewski.github.io
```

Then the website link will be:

```text
https://magdalenawischnewski.github.io/
```

## Upload these files to the repository

```text
index.html
style.css
script.js
README.md
assets/
```

Inside `assets/`:

```text
profile-photo.webp
CV_Wischnewski.pdf
publications/
```

Inside `assets/publications/`:

```text
publication-01.pdf
...
publication-25.pdf
```
